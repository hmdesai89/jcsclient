NAME
      describe-network-interfaces - Describes one or more of your network interfaces.

SYNOPSIS
      describe-network-interfaces [--network-interface-id <value>]
      describe-network-interfaces [help]

DESCRIPTION
       Describe all network interfaces in your account or specify single network
       interface by providing network-interface-id.This includes interfaces created
       explicitlly by user and interfaces created at the time of run-instances.

       Mandatory arguments: None


OPTIONS :
       --network-interface-id specify network interface id to be be described

EXAMPLES :
       jcs vpc describe-network-interface --network-interface-id eni-11112222
